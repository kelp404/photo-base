# photo-base


## Installation
**Clone**
```bash
$ git clone git@github.com:kelp404/photo-base.git
$ cd photo-base
```

**Install go packages**
```bash
$ go get github.com/gin-gonic/gin
$ go get github.com/satori/go.uuid
$ go get github.com/nfnt/resize
```

**Install node modules**
```bash
$ npm install
```

**Run develop server**
```bash
$ npm run dev
# go to browser type localhost:3000
```
